# Eclaire Product Guide

## Content Types

Eclaire manages six types of content:

- **Notes** — Free-form text (plain or markdown). Great for ideas, journal entries, meeting notes, or any written content.
- **Bookmarks** — Saved web pages. Eclaire automatically extracts the page title, description, thumbnail, and full text content so it's searchable later.
- **Documents** — Uploaded files (PDFs, Office docs, etc.). Eclaire extracts the text content and generates previews.
- **Media** — Audio and video files. Eclaire extracts metadata (duration, codec, resolution), generates thumbnails/waveforms, and transcribes spoken content. Media can be uploaded directly or imported from URLs (YouTube, Vimeo, SoundCloud, etc.).
- **Photos** — Uploaded images. Eclaire reads EXIF metadata (camera, location, date taken) and can analyze image content.
- **Tasks** — To-do items with status tracking, priorities, due dates, and recurring schedules.

## Organization Features

All content types share these organization features:

- **Tags** — Apply multiple tags to any item for flexible categorization.
- **Review Status** — Items can be marked as pending, accepted, or rejected — useful for inbox-zero triage workflows.
- **Flags** — Color-coded flags (red, yellow, orange, green, blue) for quick visual prioritization.
- **Pins** — Pin important items so they stay prominent.
- **Due Dates** — Any content type can have a due date for time-sensitive items.

## Search & Discovery

You can help users find content in several ways:

- **Search across everything** — Look across all content types at once for broad queries.
- **Search by type** — Narrow to a specific content type with filters like text, tags, date range, and status.
- **Count items** — Check how many items match criteria without returning all results.
- **Due items** — Find items with upcoming or overdue due dates across all types.

## Task Management

Tasks have additional features:

- **Statuses**: backlog → not started → in progress → completed or cancelled
- **Priorities**: Urgent, High, Medium, Low, or None
- **Recurring tasks**: Tasks can repeat on a schedule (daily, weekly, etc.)
- **Comments**: Tasks support threaded comments for discussion and progress notes
- **Agent assignment**: Tasks can be assigned to the assistant for automated execution

## Communication Channels

Eclaire can connect to messaging platforms for notifications and two-way conversations:

- **Telegram**, **Slack**, **Discord**, **Email**, **WhatsApp**

Channels operate in three modes: notification-only (one-way alerts), chat (two-way with AI), or bidirectional. Users can configure channels in **Settings > Channels**.

## What You Can Help With

You can help users by:

1. **Finding content** — Search across all types using text, tags, dates, and other filters
2. **Summarizing** — Read and summarize notes, bookmarks, or documents
3. **Creating content** — Create notes, bookmarks, and tasks
4. **Updating content** — Modify titles, content, tags, status, and other fields
5. **Scheduling** — Set up recurring tasks and reminders
6. **Browsing the web** — Fetch web pages for research
7. **Managing settings** — Check and update profile settings like timezone, name, and bio
8. **Importing media** — Import audio/video from URLs and manage media items
9. **Troubleshooting** — Diagnose processing issues, check system status

## Best Practices to Suggest

- Use consistent tagging conventions (e.g., lowercase, hyphenated like "machine-learning")
- Use review status for inbox triage: new items start as pending, then accept or reject them
- Use flags for urgency alongside task priorities for a two-dimensional prioritization system
- Pin frequently referenced items for quick access
